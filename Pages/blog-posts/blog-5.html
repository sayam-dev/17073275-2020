CSS animation & transitions

AIMS & OBJECTIVES

To implement different types of CSS animations and transitions throughout the portfolio to make the portfolio more appealing for the users, initially improving user experience overall. To achieve this I will use CSS animation and transition properties.


Definition

In one to create CSS animation, the person is required to add style properties to the element they are wanting to animate for appearance purposes and then add animation properties, which typically include: duration, timing, timing-function, delay, direction, fill-mode, play-state and iteration-count (CSS animation Property, n.d.).

Once that is done, you would be required to describe the appearance, which is achieved by giving keyframes. Keyframe are needed in order to describe how the animated element should render at a given time also, since the in-CSS animations timing is required to conduct animations, keyframes are required to use percentage to indicate time during the animation sequence (Using CSS animations, n.d.).
In the animation process 0% typically indicated the beginning of the sequence whereas 100% indicated the ending of the sequence (Cope, 2019).

Many websites use CSS animation on a smaller scale for example, for creating a hamburger menu, or animated navigations text etc. So, I thought of increasing the standards and use CSS animation and transitions to create a layout for my portfolio in order to increase user experience. The layout would consist of a wave type shape that would be constantly moving and would be placed near the margin of the screen, so I can place the content in the middle of it, which also improves accessibility.

The reason for doing this project is because I consider myself to be an amateur at using CSS, as I already know how to use CSS in a smaller scale. In the portfolio you will be able to see a hamburger menu constructed using pure CSS without using JavaScript. I also have used CSS transitions throughout the portfolio for user experience purposes. Therefore, I wanted to go beyond my comfort zone and take on a bigger project by creating the layout and using CSS animation throughout the portfolio using CSS animations and transitions.

Tools and Technologies

To set this up I used CSS animation which consists of two mechanisms which one of them allows you to style component, typically describing the CSS animation and the other is a pair of keyframes. Keyframes allows the users to usually indicate the beginning and the end of the animation; also allowing possible intermediate waypoints, therefore, making endless opportunities to generate animations overall.

How

So, as I wanted my content to be mostly in the middle of the page for the ease of viewing purposes for the users That also meant that there would have been too much whitespace alongside the margin of the screen, therefore, I wanted some shape to take place near the margin, to get rid of the extra whitespace .The animated shape that I wanted alongside the margin was to look like a shape of a circle.

The 1st step I took was to create a class called circle-two. Then I gave some initial styling properties to create the circle. I did this by applying the border attribute and giving it a colour , which in this case it’ a white. Then I applied border-radius and gave it a value of 50% which converts shape from a square to a circle. I also gave it a position attribute of absolute , which allows you to position your element on the page wherever you want, without affecting other element.

See the Pen anim & transi step 1 by sayam ajmal (@sayam-web) on CodePen.

I was able to change the element’s default position by using the helper properties (top and left)(Eygi, 2018). Therefore, I increased the size of the shape by increasing the value of the border and changed the values of top and left to align where the extra space is on the left.

See the Pen anim & transi part 2 by sayam ajmal (@sayam-web) on CodePen.

After the shape had been constructed I realised why not make the shape stay in its fixed position and make the social media icons to appear on the shape, which will allow users to constantly see the social media icons and make users an easier way of contacting me as it’s always present front of the user. I did this by then moving my class in the header section as the header section already had fixed attribute applied to it.

See the Pen anim & transi part 3 by sayam ajmal (@sayam-web) on CodePen.

After this it was time to animate the shape.so I named it spin. And applied it to the element. I also gave the animation property linear is because I want the animation to have the same speed from the start to end and also gave it infinite property as it makes the animation to go on forever, hence why it stands for infinite.

keyframes are required to use percentage to indicate time during the animation sequence but I ran into a problem as I noticed that the shape wasn’t moving at all. Even when changing the time duration , nothing seemed to be moving, so I figured out the problem, it was due to the shape being a perfect circle (50%).

See the Pen anim & transi part 4 by sayam ajmal (@sayam-web) on CodePen.

So as soon as I changed the border radius to 40%, the shape started moving and formed a wave animation, which I liked even more.

See the Pen anim & transi part 5 by sayam ajmal (@sayam-web) on CodePen.

I also wanted the animation to start at 60% and end at 100% because as I was inserting different values for the beginning of the animation, 60% seemed to make the movement of the animation much smoother compared to putting a value of 0%.

I also gave the overflow property with the value of hidden, to have a better control of the layout to clip any thing that is going over the height or the width perspectives of the screen.

This is the end result.
